# Sentiment-analysis-on-marketplace-product-reviews
Sentiment analysis on marketplace product reviews using NLP and Logistic Regression. Classifies reviews as Positive or Negative using TF-IDF vectorization and NLTK text preprocessing.
🔍 What I built:
- Created a sample marketplace review dataset (Positive/Negative labels)
- Cleaned text — lowercasing, removed special characters, stopword removal
- Extracted features using TF-IDF Vectorizer
- Trained a Logistic Regression classifier
- Evaluated using Accuracy & Classification Report
- Built a live prediction function for any new review
